## Цветомузыка на адресных светодиодных лентах

###
В проекте используется микроконтроллер **ATmega328p**. <br>
На данный момент идёт разработка программы.

###
В каталоге **source** находятся исходные файлы **(.c)**, **(.S)** и заголовочные файлы **(.h)**. <br>

Драйвер управления адресными светодиодными лентами **"addr_leds_driver.S"** написан на **ассемблере**, поскольку нужно точно контролировать время (каждый такт) включения/выключения выхода управления светодиодными лентами. <br>.

В каталоге **output** находятся результаты компиляции и сборки программы, а именно: обьектные файлы **(.o)**, файлы форматов **(.elf)** **(.hex)** **(.bin)** <br>
